# Mofiz Uddin Sagor — Portfolio Website

Personal portfolio website for **Mofiz Uddin Sagor**, a Flutter Developer with 2.5+ years of professional experience building cross-platform mobile apps for Android and iOS.

Live site: [sagorahamed1.github.io](https://sagorahamed1.github.io)

## Overview

A fully responsive, single-page portfolio built with vanilla HTML, CSS, and JavaScript. No frameworks or build tools required — just open `index.html` in a browser.

## Sections

- **Hero** — Introduction, headline, and call-to-action buttons
- **Stats** — Years of experience, published apps count, and happy clients
- **About** — Background summary and professional overview
- **Skills** — Categorized skill cards covering Flutter, state management, backend, payments, authentication, DevOps, and more
- **Experience** — Work timeline (Software Engineer at Spark Tech Agency, 2023–Present)
- **Projects** — Featured published apps with App Store and Play Store links
- **Contact** — Email, phone, location, and social links

## Running Locally

No build step needed. Simply open the file in your browser:

```bash
open index.html
```

Or serve it with any static file server:

```bash
npx serve .
# or
python3 -m http.server 8080
```
